# Filter Lists

This directory contains pre-installed filter lists for the content blocker.

## Default Lists

- **EasyList** — Primary ad blocking list
- **EasyPrivacy** — Tracker blocking list
- **Fanboy's Social Blocking List** — Social widget blocking
- **Fanboy's Annoyances** — Cookie notices, popups, etc.

## Format

Filter lists use the Adblock Plus / uBlock Origin syntax.
See: https://adblockplus.org/filter-cheatsheet

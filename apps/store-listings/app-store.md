# App Store Listing (iOS + macOS)

## App Name
Veil — Content Blocker

## Subtitle (30 chars)
Ads, Trackers & Widgets

## Description
Veil is a powerful content blocker for Safari on iPhone, iPad, and Mac. One app for all your Apple devices — install it and browse the web without distractions.

⚡ HIGH PERFORMANCE:
• Built on a token-bucket engine processing 1.7M requests/sec
• Sub-microsecond matching latency (0.6μs per request)
• Loads 300K rules in under 250ms
• Cached cold start in <50ms for instant protection
• Optimized for Apple Silicon and A-series chips

🚫 COMPREHENSIVE BLOCKING:
• Ads — banners, video ads, pop-ups, interstitials
• Trackers — analytics, fingerprinting, cross-site tracking
• Social widgets — Facebook, Twitter, Instagram, LinkedIn, VK
• Annoyances — cookie banners, newsletter pop-ups, chat widgets
• Anti-adblock bypass with 116 scriptlets

🔒 PRIVACY FIRST:
• $removeparam — strips tracking parameters (utm_*, fbclid, gclid) from URLs
• CNAME uncloaking — detects trackers hiding behind first-party DNS
• No data collection — everything runs on your device
• No accounts, no servers, no tracking

👥 SOCIAL WIDGET PLACEHOLDERS:
• Blocked social embeds show a clean placeholder
• One-click to load when you choose to interact
• No social tracking until you explicitly allow it

📊 FEATURES:
• Per-tab and daily/weekly blocking statistics
• Element picker — create custom rules by tapping elements
• Whitelist for trusted sites
• Custom user rules with full ABP/uBO/AdGuard syntax
• 30+ filter modifiers for precise control
• Extended CSS selectors for complex element hiding
• Auto-learning engine adapts to new ad patterns
• Engine serialization for instant cold start

☁️ SYNC:
• iCloud synchronization across all your Apple devices
• Settings, custom rules, and whitelist sync automatically
• Conflict resolution for simultaneous edits

📋 FILTER LISTS:
• EasyList (ads)
• EasyPrivacy (trackers)
• Fanboy Social (social widgets)
• Fanboy Annoyances (cookie banners, pop-ups)
• Regional lists (RU AdList and more)
• Automatic background updates

🔧 SAFARI INTEGRATION:
• Native WebKit Content Blocker API for maximum performance
• Up to 150,000 rules per content blocker instance
• Works with Safari on iOS, iPadOS, and macOS
• Supports Safari Web Extensions for advanced features

OPEN SOURCE:
Veil is fully open source under the MIT license. Verify the code yourself — your privacy is guaranteed by transparency.
